Scratch boost

Scratch Boost is an extension for the website Scratch that adds several new blocks.

Download this project as a .zip file Download this project as a tar.gz file

Blocks it adds:

                alert [Hello!]::extension
                
                (prompt [What is your favorite color?]::extension)
                (prompt [What is your favorite color?] with default value [Purple]::extension)

Installation

To install Scratch Boost all you have to do is drag this bookmarklet Scratch Boost. Then, when viewing a Scratch project click the bookmarklet to install Scratch Boost.
